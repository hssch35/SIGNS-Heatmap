# Paleography Analysis: KBo 1.8++ (Bentešina Treaty)

This project visualizes the distribution of paleographic variants in the Hittite treaty **KBo 1.8++** (CTH 92). 
It maps diagnostic cuneiform signs against their line numbers to identify "ductus clusters" (Hittite vs. Syro-Mesopotamian styles).

## Features
- **Ductus Scoring:** Visualizes signs based on a 3-point scale (0 = Hittite, 1 = Transitional, 2 = Foreign/Assyro-Mittani).
- **Text Integration:** Maps paleographic data points directly to the transliteration.
- **Visual Strategy Map:** Generates a scatter plot showing the "hand" of the scribe across the tablet's layout.

Data Source
Transliteration: Based on HethPort (Würzburg).

Paleography: Own dataset based on diagnostic signs (LI, ŠAB, IT, etc.).
